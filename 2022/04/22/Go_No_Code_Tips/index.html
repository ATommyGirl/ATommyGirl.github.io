Go 乱七八糟之 GoLand 不提示不高亮

YYLittleCat
2022-04-22
Go

当我正在愉快的使用 Go 编写 “Hello, World!” 时，GoLand 弹出了一个提示框、有三两个选项，说实话我没细看就随便选了一个，然后…再敲代码时，code 自动提示没有了、pkg 自动引入没有了、换行时自动缩进没有了、代码高亮没有了、方法旁边的绿色小按钮也不见了、打开 “Hello, World!” 项目居然都要转半分钟的菊花…

…哭笑不得…

跟同事鼓捣了半天 File Encodings，发现并不是编码格式的事儿。就在我要重装 GoLand 时，在偏好设置里发现了一个很突兀的 2…

是的，Preferences -> Editor -> File Types，选中 Go files ，发现 Go 文件的后缀 *.go 里多了一个 2，然后 Text 类型文件的后缀里多了一个 *.go …
所以把 Text 中的 *.go 删掉，把 Go files 中的 *.go2 改为 *.go 就好了。

破案了破案了，至于为什么…小咪表示也不知道…😂
就…还是…哭笑不得…
